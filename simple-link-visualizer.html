Simple link visualizer

How to set it up

Two methods to set up the bookmarklet:

  • drag the link to your bookmark bar;
  • or manually create a new bookmark that points to the link URL.
    Right-clic on the link, copy the link address, open the browser bookmarks and create a new bookmark, paste in the URL field what was previoulsey copied.

How to use it

Go to the page you want to test and then to the bookmarked page.

Results

Links on the page that include "http" (so external links for sure) will be displayed prominently.
Links in empty tags will also be displayed with extra emphasis.

Good luck! Hope your links are as expected and if not fix them!
